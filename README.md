# QuestPass 

Turn your layover into a time-safe and fun city quest to discover hidden gems in Singapore and beyond.

## What it does

QuestPass helps travelers with short Singapore layovers choose themed local quests, solve clues, unlock real places, collect badges, and return to the airport on time.

## Core experience

- Enter flight time
- Select available layover window
- Choose a themed quest
- Solve location clues
- Navigate to real-world stops
- Unlock places when physically nearby
- Collect XP, badges, and passport stamps

## Sample quests

- Old-School Singapore
- Hidden Hawker Gems
- Heritage Shophouse Quest
- Date in Tiong Bahru
- Vegetarian-Friendly Lunch Quest

## GrabMaps integration notes

GrabMaps should be used for:

- Place search
- Route distance
- Route ETA
- Airport return ETA

Important:

- Use straight-line distance only for unlock radius.
- Use GrabMaps route distance for actual travel distance and ETA.
- Keep demo mode and fallback coordinates working.
- Do not commit API keys directly into GitHub.
